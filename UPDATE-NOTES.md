## 🛠️ 使用方式

### 生成完美代码
```bash
# 使用模板创建新文件
cmx-skills generate --template perfect-header --name MyClass
cmx-skills generate --template perfect-source --name MyClass

# 使用提交模板
cmx-skills commit --type feat --scope network --issue 42
```

### 代码质量检查
```bash
# 检查代码是否符合完美标准
cmx-skills check --file src/myclass.cpp --template perfect

# 获取改进建议
cmx-skills suggest --file src/myclass.cpp
```
